## Features

* Replaces specific strings displayed in Discord with custom text.
* Works only on your local Discord client.
* Does not modify, edit, send, or delete actual Discord messages.
* Does not affect how messages appear to other users.
* Supports multiple replacement rules.
* Supports case-sensitive replacements.
* Supports case-insensitive replacements.
* Automatically applies replacements to newly loaded messages.
* Monitors Discord UI updates and re-applies replacements when needed.
* Restores the original displayed text when the plugin is disabled.
* Can be limited to message content only.
* Avoids replacing text inside input fields, textboxes, code blocks, scripts, and styles.
* Runs as a lightweight BetterDiscord plugin in a single JavaScript file.
